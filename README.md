# Quantis | Compound Your Legacy

Institutional-grade asset management landing page — a Swiss-inspired, technical design system with scroll reveals, animated counters, and a distinctive visual identity.

## Tech Stack

- **HTML** — Semantic markup
- **Tailwind CSS** — Via CDN (Space Grotesk, Inter, JetBrains Mono)
- **Iconify** — Icon set
- **Vite** — Dev server with HMR

## Quick Start

```bash
# Install dependencies
npm install

# Run local dev server (opens http://localhost:3000)
npm run dev

# Build for production
npm run build

# Preview production build
npm run preview
```

## Design System

| Token   | Hex       | Usage              |
|---------|-----------|--------------------|
| q_black | `#0A0A0A` | Primary text, CTAs |
| q_graphite | `#5C5C5C` | Secondary text     |
| q_acid  | `#FF3800` | Accent, highlights |
| q_bone  | `#F4F4F0` | Background         |
| q_white | `#FFFFFF` | Cards, contrast    |
